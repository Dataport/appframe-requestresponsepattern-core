Class ActionHandler<TRequest, TResponse>

Ermöglicht es einen Handler vereinfacht durch eine Action zu implementieren. Macht im Wesentlichen bei UnitTests Sinn, um dort die Mockups bereitzustellen und on-the-fly als RuntimeListeners zu registrieren.
